# 🔍 NetScope

NetScope is a Python-based network reconnaissance and analysis tool that helps users gather essential information about a target host or domain from a single interface.

## 🚀 Features

- 🌐 DNS Lookup
  - Retrieve IP addresses associated with a domain.
  - Display hostname information.

- 🔎 Port Scanning
  - Scan common ports such as:
    - 22 (SSH)
    - 80 (HTTP)
    - 443 (HTTPS)
  - Identify open services running on a host.

- 📡 Reverse DNS Lookup
  - Convert an IP address back to its hostname.

- 📋 WHOIS Information
  - Fetch domain registration details.
  - Display registrar and registration dates.

- 📝 Scan History
  - Maintain a history of previous scans.

- 🖥️ User-Friendly GUI
  - Built with Python's Tkinter library.
  - Simple and interactive interface.

## 🎯 Purpose

NetScope is designed for:

- Cybersecurity students
- Network administrators
- Ethical hackers
- Beginners learning network reconnaissance techniques

It provides a centralized platform to perform basic reconnaissance tasks without relying on multiple tools.

## 🛠️ Technologies Used
